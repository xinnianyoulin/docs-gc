# 通用组件列表

# GridCells

定制栅格组件，vue-grid-layout的简单包装

props

  • layout : Object default:{}

    一个具有vue-grid-layout标准格式的栅格配置文件

    const layout = {
  colNum: 24,
  rowNum: 24,
  isDraggable: true,
  isResizable: true,
  data: [
    {
      x: 0,
      y: 0,
      w: 6,
      h: 8,
      i: "0",
      component: yourComponent,
      config: {} // 当前组件会接收一个props.config
    },
    ...
  ]
};

  • margin : Array default:[8,8]

    栅格模块间距

events

事件触发后返回 layout 对象数据组

  • @layout-created="layoutCreatedEvent"

    在组件 beforeCreated 生命周期钩子上激活

  • @layout-before-mount="layoutBeforeMountEvent"

    在组件 beforeMount 生命周期钩子上激活

  • @layout-mounted="layoutMountedEvent"

    在组件 mount 生命周期钩子上激活

  • @layout-ready="layoutReadyEvent"

    当挂钩上的所有操作完成时发出声响

  • @layout-updated="layoutUpdatedEvent"

    每次布局完成更新并重新计算所有网格项的位置

methods

  • openDrag 开启拖拽
  • openResize 开启尺寸缩放
  • closeDrag 关闭拖拽
  • closeDrag 关闭尺寸缩放

# DataTable

表格组件

props

  • config [Object]

    表格配置项信息，默认配置

    const config = {
  header: [], // 见header
  pageSize: 20, // 分页
  options: {
    // el-table中所有attribute
    size: "small",
    border: true,
    height: "100%",
    headerCellStyle: {
      fontSize: "15px",
      color: "#78899d",
      fontWeight: "normal",
      backgroundColor: "#f3f6f9"
    }
  }
};

// header 为表头配置信息,没有默认项，基本配置如下
const header = {
  type: "index/slot",
  // 序号/自定义项（在默认slot中接收slot-scope="{row,column}"）分别为当前行及列数据,
  // 如果不指定即为普通项
  label: "name", // 表头
  value: "key", // 对应字段
  width: 100, // min-width
  fixed: true,
  format: "yyyy-MM-dd", // （时间格式化）存在时，默认为时间项
  align: "center"
};

  • data [Array] default:[]

    数据源

slot

  • default: 当 header 中有 type = slot 项时生效，slot-scope="{row,column}"，用来自定义每个单元格内容

# DataChart

图表组件，基于 v-charts（为方便起见，只提供常见的线、饼图、柱状图）
详细 options 可以参见echarts

props

  • config [Object]

    表格配置项信息，默认配置

    const config = {
  // 曲线类型，可选值['line', 'pie', 'histogram']
  type: "line",
  maps: {
    series: "name", // 数据源系列名字段
    xAxis: "name", // 数据源维度
    yAxis: "val" // 数据源指标
  },
  //  v-charts's chartSettings
  settings: {},
  // echarts's options & v-charts's chartExtends
  options: {
    grid: {
      bottom: 20
    },
    series: {
      center: ["50%", "60%"],
      radius: "60%",
      barMaxWidth: 25
    }
  }
};

  • data [Array] default:[]

    曲线图的数据格式 data 默认为数组，每个项分别指向每个系列

    // 两个系列，config.maps.series对应为水质水情系列字段，xAxis、yAxis分别对应time、val
const demo = [
  {
    name: "水质",
    data: [{ time: "123", val: 1 }, { time: "123", val: 2 }]
  },
  {
    name: "水情",
    data: [{ time: "123", val: 1 }, { time: "123", val: 1 }]
  }
];

# PickTree

  • config [Object] 参见树形项

  • @rows 返回一个对象，包含所选项数据

# CustomSeed

集成通用组件以及具备配置对应图表过滤项的简单业务组件，通过接口及过滤项配置，自动完成接口处理操作

props

  • config [Object]

    主要配置：

    const config = {
  id: 1, // id
  type: 'data-chart', // 所使用子组件
  title: '434Km气温趋势线', // 主标题
  subTitle: 'subtitle', // 副标题
  debug: true, // 是否开启调试，用于获取filterBar等返回的数据信息（console.log)
  titleTemplate: { // 开启自定义动态标题
    template: '$temp1$$temp2$$temp3$动态标题内容',
    format: 'MM月', // 全局格式化
    rules: [
      ...
      { // template中需要替换的key
        key: '$temp1$',
        // filterBar.config.search中所配置的对象的类型加上相应的value字段，建议开启debug，
        // 查看过滤项返回的字段来取相应的值，取值方式为js中对象取值，如：group.name.areaname
        keyValue: 'time1-start'
      },
      {
        key: '$temp2$',
        keyValue: 'f_month',
        format: 'MMyue' // 单个设置format，优先级更高
      },
      {
        key: '$temp3$',
        keyValue: 'group.name.areaname'
      }
      ...
    ]
  },
  icsv: true, //是否导出csv （暂未添加）
  ipng: true, //是否导出图片 （暂未添加）
  url: 'api', //组件接口地址
  search: [], // 组件筛选项配置 参见FilterBar
  config: {}, //组件自定义配置 如表格配置
  dataMaps: function(data) {
    return data
  }, // 针对接口返回的字段不满足子组件要求时，对所请求的数据格式化方法
  theme: 'default' //主体关键字
}

    提示：关于 search 数组对象中默认项 `default` 设定

    普通非嵌套项 default 绑定的是接口请求默认参数以及控件默认选中类型， 嵌套项 daterange、group 的 default 绑定的仅是接口请求默认参数， 控件默认选中需要在嵌套数组 range、groups 内定义 default 来初始化（最好与外部 default 相一致）
    search配置项参见筛选项配置

    events

  • @updated: 返回一个对象，出现过滤动作后触发，该对象为过滤组件修改后的过滤参数与传入的 config 相合并（合并至 default 中）

# FilterBar

过滤面板组件，置于 popover 中，接收一个与CustomSeed相同的配置项 config，加载相应的过滤组件

props

  • config [Object] 参见CustomSeed.config

events

  • @filter 返回一个对象，包含当前所选过滤项所配置的接口参数
  • @rows 返回一个对象，其中包含所有过滤项原生字段项的内容，可以用作自定义内容

slot

  • default: 触发面板 popover 的 reference，默认是一个 setting 图标
